# Customer Retention & Churn Analysis

## 📊 Project Overview
This project analyzes customer data to understand churn patterns and retention trends in a subscription-based business.

## 🛠️ Tools Used
Python, Pandas, Matplotlib, Seaborn

## 📈 Analysis Performed
- Churn count analysis
- Churn by gender
- Churn by contract type
- Monthly charges vs churn
- Tenure vs churn

## 🔍 Key Insights
- Customers with month-to-month contracts have higher churn.
- Higher monthly charges lead to increased churn.
- Long-term customers show better retention.
- Gender has minimal impact on churn.

## 💡 Recommendations
- Encourage long-term contracts with discounts.
- Reduce pricing for high-paying customers.
- Improve customer engagement strategies.
- Offer loyalty benefits to retain customers.
